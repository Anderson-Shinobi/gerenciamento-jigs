# gerenciamento-jigs

## Descrição
Projeto com uma página HTML para visualização de informações de JIGs.

## Finalidade
Disponibilizar uma interface simples de consulta para uso interno, com foco em integração com sistema baseado em Excel VBA.

## Estrutura do repositório
- `README.md`: documentação do projeto.
- `visualizar_jig.html`: página de visualização das informações de JIGs.

## Como abrir no navegador
1. Acesse a pasta do projeto.
2. Abra o arquivo `visualizar_jig.html` diretamente no navegador.

## Dependências
Este projeto não exige instalação de dependências.

## Tecnologias usadas
- HTML5
- CSS3

## Status do projeto
Em uso interno.

## Observação
Projeto destinado a uso com sistema baseado em Excel VBA.
